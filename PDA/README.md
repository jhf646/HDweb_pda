# PDA仓储管理系统

基于Web的PDA设备入库绑定系统，适配工业级PDA设备扫码操作。

## 功能特性

- ✅ 用户登录认证
- ✅ 托盘号扫描输入
- ✅ POST请求接口调用
- ✅ PDA设备触摸屏适配
- ✅ 扫码器自动识别
- ✅ 高对比度工业模式

## 接口说明

### 查询托盘信息

- **URL**: `http://192.168.2.53:20000/QueryPalletInfo`
- **Method**: POST
- **Request Body**:

```json
{
    "PalletID": "20220812B796"
}
```

## PDA适配说明

### 扫码器配置

系统支持两种扫码方式：

1. **自动监听模式**：监听全局键盘输入，自动识别扫码数据
2. **手动输入模式**：在输入框中手动输入或扫描

### 设备测试

建议在PDA设备上测试以下功能：

- 触摸屏操作流畅性
- 扫码器识别准确性
- 网络请求稳定性

## 使用说明

1. 将项目文件部署到Web服务器
2. 在PDA浏览器中打开 `index.html`
3. 使用默认账号登录：
   - 用户名: `admin`
   - 密码: `123456`
4. 扫描托盘二维码或手动输入托盘号
5. 系统自动发送查询请求并显示结果

## 开发说明

系统采用原生JavaScript开发，无需框架依赖，确保在各类PDA设备的浏览器中稳定运行。

## 安全提示

- 生产环境请修改默认登录凭证
- 建议启用HTTPS加密传输
- 对接企业认证系统
